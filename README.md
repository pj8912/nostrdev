## Features
- Generate Keys
- Convert Keys
- Publish a Note
- Add Relays

## Installation

- **Clone the repo**

 ``` 
 git clone https://github.com/pj8912/nostrdev.git
 ```
 - **Create Database**
 ```shell
 ~/$ cd nostrdev
 ~/nostrdev$ php create_database.php
 ```
- **composer install**
```shell
~/nostrdev$ composer install
```

### Start server
```shell
php -S localhost:4000
```
